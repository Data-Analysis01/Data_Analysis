# Power Usage Energy Analysis Dashboard

## Project Overview

This project focuses on analyzing power usage and energy consumption data through advanced data analytics and interactive dashboard development. The objective is to convert raw energy datasets into meaningful visual insights that support efficient monitoring and decision-making.

## Features

### Data Preparation

* Data Cleaning and Validation
* Missing Value Handling
* Data Transformation and Standardization
* Data Import and Integration

### Data Modeling

* Fact Table and Dimension Table Design
* Relational Data Modeling
* Structured Analytical Framework

### Advanced Analytics

* Pivot Table Analysis
* Dynamic KPI Calculation
* Trend Analysis
* Energy Consumption Monitoring

### Dashboard Development

* Interactive Dashboard Interface
* Pivot Charts and Visualizations
* Slicers and Filters
* Two-Way Navigation Between Dashboard and Tabular Sheets
* User-Friendly Reporting Experience

## Tools Used

* Microsoft Excel
* Power Query
* Pivot Tables
* Pivot Charts
* Advanced Excel Functions
* Data Modeling Techniques

## Key Skills Demonstrated

* Data Cleaning
* Data Transformation
* Data Analysis
* Data Visualization
* Business Intelligence
* Dashboard Design
* Fact & Dimension Modeling
* Report Automation
* Analytical Problem Solving

## Business Value

The dashboard provides actionable insights into power consumption behavior, helping stakeholders identify usage trends, monitor performance indicators, and improve operational efficiency through data-driven decision-making.
